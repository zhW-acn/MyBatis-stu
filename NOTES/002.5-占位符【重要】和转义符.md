占位符
-

1. ${}：它的底层是用Statement来处理SQL语句的，拼接字符串方式有注入风险
2. #{}：他的底层是用PreparedStatement来处理，通过占位符`?`
    - #{arg0}:参数列表第一个
    - ATTENTION：3.4.4版本之后用#{arg0},#{arg0} 3.4.4版本之前用#{0},#{1} ,#{2}

转义符
-

|     | 转义后    |
|-----|--------|
| <   | \&lt;  |
| <=  | \&le;  |
| \>  | \&gt;  |
| \>= | \&ge;  |
| &   | &amp;  |
| "	  | &quot; |
| ’	  | &apos; |

或使用`<![CDATA[]]>`忽略，是xml中通用的解决方案
